# Questions

A personal collection of small programming projects, experiments, random ideas, and problem-solving exercises made in my spare time.

There is no strict theme for this repository — some files solve questions, some exist just for the sake of building something, testing an idea, or practicing syntax.

The only consistent rule is that each project is kept as a single-file implementation whenever possible.

## Goals

- Practice programming regularly
- Explore different languages
- Build things without overcomplicating them
- Keep an archive of small standalone code ideas

## Notes

Languages may vary depending on mood, curiosity, or purpose.
Code style may also change over time.
